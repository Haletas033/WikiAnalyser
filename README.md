# WikiAnalyser

## About:
WikiAnalyser is a lightweight C program that 
allows you to easily and quickly
gather data from every Wikipedia article, compile it, 
and turn it into interactive
interesting, useful, or random graphs/spreadsheets.
All in under 100kb (excluding the 110gb Wikipedia dump) (:

## Current Features (*not much as it is a new project*):
### Cleanup Features:
* Set what containers, strings, and characters you want removed
* Single-pass efficient clean up algorithm for large files
* Memory safe

## Upcoming Features:
### Cleanup Features:
* Presets for Unwanted characters, strings, and containers
* strlen caching
* GPU acceleration
* Option to clone cleaned file into new file
### Parsing Features:
* Define your own struct for every article
* Write functions to explain how to fill these fields
* Save data onto disk
### Analysis Feature:
* Multiple graph types including graph theory ones
* Interactive graphs
* Download graphs as images or animations
* save data to a spreadsheet for use in Excel or Google Sheets
* customize the style of graphs to meet your needs
